#Flask Setup

## Windows
1) Create an environment
'''
$ cd MilkyWayRepairs
$ py -3 -m venv .venv
'''
2) Activate the environment
'''
$ .venv\Scripts\activate
'''

3) Install Flask
'''
$ pip install Flask
'''

## MacOS / Linux

1) Create an environment
'''
$ cd MilkyWayRepairs
$ python3 -m venv .venv
'''
2) Activate the environment
'''
. .venv/bin/activate
'''

3) Install Flask
'''
$ pip install Flask
'''
